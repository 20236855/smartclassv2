========================================
Whisper 视频转录服务 - 快速启动
========================================

【作用】
将视频中的语音转换成文字，用于知识点抽取

视频 → 语音识别 → 文字 → AI分析 → 知识点


【快速启动 - 3步】

步骤1：安装依赖（首次使用，约2分钟）
--------------------------------------
cd C:\smartclassv2\RuoYi-Vue-master\whisper-service
pip install openai-whisper flask flask-cors


步骤2：启动服务（每次使用）
--------------------------------------
python whisper_server.py


步骤3：验证成功
--------------------------------------
浏览器访问：http://localhost:5000/health

看到这个表示成功：
{
  "status": "ok",
  "service": "whisper-transcription",
  "loaded_models": ["tiny"]
}


【成功标志】

看到这个输出表示启动成功：
--------------------------------------
Loading model: tiny
Model loaded successfully
 * Running on http://0.0.0.0:5000
--------------------------------------

⚠️ 重要：保持窗口打开，不要关闭！

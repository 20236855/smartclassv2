## 📖 题目示例

### 单选题示例

**题目3001**：HTML的全称是什么？
- A. HyperText Markup Language ✅
- B. High Tech Modern Language
- C. Home Tool Markup Language
- D. Hyperlinks and Text Markup Language

**正确答案**：A  
**解析**：HTML是HyperText Markup Language的缩写，是网页开发的基础标记语言。

### 多选题示例

**题目3021**：以下哪些是HTML5新增的语义化标签？（多选）
- A. `<header>` ✅
- B. `<footer>` ✅
- C. `<nav>` ✅
- D. `<div>`

**正确答案**：A,B,C  
**解析**：HTML5新增了多个语义化标签，包括`<header>`、`<footer>`、`<nav>`、`<article>`、`<section>`等。

### 判断题示例

**题目3026**：HTML是一种编程语言。
- A. 正确
- B. 错误 ✅

**正确答案**：B（错误）  
**解析**：HTML是标记语言，不是编程语言。编程语言具有逻辑控制、变量等特性。

## 🎓 使用建议

1. **教师端**：
   - 可以根据需要修改作业的开始和结束时间
   - 可以调整题目分值和顺序
   - 可以添加更多题目到作业中

2. **学生端**：
   - 建议按照难度顺序完成作业（从简单到困难）
   - 注意多选题的评分规则
   - 合理安排答题时间

3. **系统管理员**：
   - 定期备份题库数据
   - 监控作业提交情况
   - 分析学生答题数据
